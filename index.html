01
No model router — everything hits Opus

By default, OpenClaw sends all tasks to claude-opus-4-6. It's the most capable model, at $15/MTok input. Simple tasks like formatting, lookups, and classification cost the same as complex reasoning. They shouldn't.

02
Context windows stay open too long

OpenClaw keeps your full conversation context in memory across tasks. Good for coherence. Terrible for cost. After 2–3 hours of runtime, every new call includes tens of thousands of tokens of stale context you're paying to re-send.

03
Tool calls aren't batched

Sequential tool invocations mean sequential API round-trips. A workflow that calls 8 tools one-by-one sends 8× the overhead tokens. The skill file batches them. Average reduction: ~40% fewer tokens on tool-heavy workflows.
