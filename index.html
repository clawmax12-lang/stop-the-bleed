# Your agent is using a sledgehammer to crack every nut.
01
No model router — everything hits Opus

By default, OpenClaw sends all tasks to the highest-tier model. Simple lookups, formatting calls, and heartbeat checks cost the same as complex reasoning. Opus is $15/MTok. Haiku is $0.50. They don't need to.

02
Context windows stay open too long

OpenClaw keeps your full conversation history in memory across tasks. After 2–3 hours, every new API call includes tens of thousands of stale tokens you're paying to re-send on every single request. Setting context_ttl drops this immediately.

03
Heartbeats are burning money while you sleep

A heartbeat every 15 minutes is 2,880 idle API calls per month — each one carrying your full SOUL.md + AGENTS.md overhead. Drop the frequency to 2-hour intervals and switch the model to Haiku. That single change saves ~60% of idle spend.
